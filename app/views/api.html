About the API

Shut Up, Seattle (SuS) empowers citizens to make informed decisions about where they work, play, and live. Similarly, the SuS API seeks to make data related to noise pollution as public as possible.

SuS is the first app and API that visualizes or estimates noise pollution in Seattle. This dataset is extremely useful for people who are currently moving and/or for whom noise is a concern. SuS gives users this knowledge, allowing them to move into or visit a location with their eyes and ears wide open. Given the rapid growth and expansion in Seattle today, construction and other noise is cropping up everywhere, and residents deserve to know what to expect.

At its core, the SuS API provides a standardized score based on anticipated noise levels at different locations in Seattle, determined from data provided by the city tracking some of the more common sources of noiseâ€”construction, bars, fire stations, and so forth. The score relies on the Haversine Formula and the Inverse Square Law to calculate the aggregate noise level at any given point within Seattle city limits.

Retrieve Common Noises In Seattle

To retrieve a list of all the noises in our database, currently limited to only Seattle, use the following query:

GET http://api.shutupseattle.com/noises

The response should be formatted as follows:

    [
      {
        "id":35701,
        "noise_type":"fireStation",
        "lat":47.699030679286,

Generating a Location Score

To generate a Location Score, you'll need to supply the API with latitude and longitude coordinates of your query location. For example:

GET http://api.shutupseattle.com/score?latitude=47.63619109&longitude=-122.34287009

A successful result will look like this:

    {
      "score":"F",
      "noises":[
        {
          "noise_type":"1 Demolition Site",
          "icon":"wrench",
          "details":[
            "Demolish existing single family residence subject to field inspection."
          ]
        },
        {
          "noise_type":"1 Bar",
          "icon":"glass",
          "details":null
        },
        {
          "noise_type":"2 Transit Stops",
          "icon":"road",
          "details":null
        },
        {
          "noise_type":"1 Freeway",
          "icon":"road",
          "details":null
        }
      ]
    }

Note: The API will return an error message if your query is outside of Seattle city limits. We roughly define that area as latitudes between 47.48172 and 47.734145, and longitudes between -122.224433 and -122.459696.
